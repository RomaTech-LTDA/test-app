GitHub

API de demonstração que integra todos os pacotes RomaTech com o pattern de decorators (idêntico ao .NET).

Pattern

@Controller('/users')
class UserController {
    @Get('/')
    @AiDescription('Lists all users')
    static async getAll(req, res) { ... }

    @Post('/')
    @AiTool('create_user')
    @AiDescription('Creates a new user')
    static async create(req, res) { ... }

    @Delete('/:id')
    @AiHidden()
    static async delete(req, res) { ... }
}

// server.ts
useController(app, UserController);
useAi(app);

Endpoints

MétodoRotaAI
GET/usersReadOnly (RAG)
POST/usersMCP Tool: create_user
PUT/users/:idMCP Tool: update_user
GET/productsReadOnly (RAG)
GET/products/searchMCP Tool: search_products
POST/productsMCP Tool: create_product
PUT/products/:idMCP Tool: update_product

URLs

  • API: http://localhost:3000
  • Swagger UI: http://localhost:3000/api-docs
  • MCP: POST http://localhost:3000/mcp
  • Metrics: http://localhost:3000/mcp/metrics

Stack

  • Express 5 + TypeScript
  • @romatech/orm (Entity Framework-like)
  • @romatech/swagger (zero-config OpenAPI)
  • @romatech/ai-extensions (MCP + RAG via decorators)
  • @romatech/linq (LINQ em Array.prototype)
